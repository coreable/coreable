# ChangeSubjectState

## Declaration

> changeSubjectState() {}

## Description

> The method used by managers to change the state of a subject to 1, 2 or 3

## Preconditions

> The user logged in must be a manager
> 
> The user must be a manager of that subject
>
> The state must be either 1, 2 or 3

## Parameters

> | Field | Type  | Required? |
> |:-:|:-:|:-:|
> | subject_id | String | Yes |
> | state | Number | Yes |

## Shell 

```graphql
mutation {
  changeSubjectState(subject_id: "", state: 0) {

  }
}
```

## Example Key

> | Icon | Meaning  |
> |:-:|:-:|
> | # field  | Field always returns null |
> | # field {}  | Subquery always returns null |
> | # =>[Class]  | Subquery returns a list  |
> | # =>{Class}  | Subquery returns an object  |
> | # //  | Comment  |

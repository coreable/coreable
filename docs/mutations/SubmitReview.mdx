# SubmitReview

## Declaration

> submitReview() {}

## Description

> The method used to submit a review coming from the logged in user, targeted at one of their team members
 
## Pre Conditons

> The user must be logged in
>
> The user must have a common team with the user being reviewed
>
> There must be only 1 review completed per stage on a user

## Parameters

> | Field | Type  | Required? |
> |:-:|:-:|:-:|
> | inviteCode | String | Yes |

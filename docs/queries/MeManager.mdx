# me (as Manager)

## Declaration

> me {}

## Description

> The **me** query is the query from the users perspective. It returns information about the logged in user.

## Pre Conditions

> The user must be authenticated
>
> The user must be a manager

## Parameters

> None

## Shell

```graphql
query {
  me {

  }
}
``` 

## Example Key

> | Icon | Meaning  |
> |:-:|:-:|
> | # field  | Field always returns null |
> | # field {}  | Subquery always returns null |
> | # => [Class]  | Subquery returns a list  |
> | # => {Class}  | Subquery returns an object  |
> | # //  | Comment  |
